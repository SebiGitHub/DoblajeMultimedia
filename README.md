# DoblajeMultimedia (Android)

## Qué es
Proyecto de práctica orientado a trabajar con contenido multimedia y “doblaje”/audio asociado (p. ej., reproducir vídeo y controlar una pista de audio o efectos).  

## Stack
- Kotlin
- Android Studio
- Multimedia: MediaPlayer / ExoPlayer / AudioTrack
- Recursos/archivos multimedia

## Features
- Reproducción de vídeo + control de audio
- Sincronización básica / selección de pista / overlay
- Gestión del ciclo de vida y liberación de recursos

## Cómo ejecutar
1. Clona el repo
2. Abre en Android Studio
3. Coloca los archivos multimedia donde espera la app
4. Ejecuta en dispositivo (recomendado si hay audio)

## Qué aprendí
- Conceptos de audio/vídeo y control desde UI
- Manejo correcto de recursos y compatibilidad
- Depuración de problemas típicos de multimedia (latencia, ciclo de vida)
